# qase
Extremely simple music playback program

## dependencies
ffmpeg(ffprobe)

SDL2

cmake

## Usage
`qase <Music file>`

## How to build
```
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .
```
